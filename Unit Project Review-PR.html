Unit Project Reflection

Doing this project was definitely a welcomed challenge for me. I have only ever made websites with platforms such as WordPress, where everything is already set up for you and you just have to plop your information in, so this was definitely new to me. I did have some challenges with figuring out file types and how to make everything show up the way I wanted it to, but I think those struggles helped me learn the process better. I would be lying if I said that I enjoyed this process, because I really did not. I felt like 90% of the time spent on this was just pure, unadulterated struggling. I do think that part of my frustration is largely due to the fact that I was working with a MacBook, which are notorious for being difficult with any file other than a PDF. As much as this project was about learning how to use programs like GitHub, it was also just about figuring out how to do all of this work with the computer I have. I am glad that this project was not the easiest for me because it really made me feel like I learned something and grew from the process.

Over all, I am pretty proud of this project. This definitely isn't the most fancy presentation, but I had to learn a lot to get to even this point. It is not very often that I feel this challenged by a project, so I was glad to face a little adversity and really push myself to grow and leave my comfort zone. Even if this isn't the best project ever, I am proud of it because it helped me remember that I can do things that seem hard at first.
